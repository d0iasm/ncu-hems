# 名古屋市立大学芸術工学部実習製作

## Tools
* Vagrant 1.9.1
* VirtualBox 4.3.28

## Getting Started

* [Download Vagrant](https://www.vagrantup.com/downloads.html)
* [Download VirtualBox](https://www.virtualbox.org/wiki/Downloads)
* `git clone https://github.com/d0iasm/ncu-hems.git`
* `vagrant up`
